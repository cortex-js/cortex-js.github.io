Hello!

In 1984 I built WinMath for the Mac, one of the first graphing calculator for modern computers.

A few years later I had the privilege of bringing the Mac UI to life at Apple, then I helped build tools for creative professionals and move the web platform forward at Adobe.

Now I'm returning where it all began: I'm building tools for scientific computing, including mathfield, a web component to edit beautifully typeset math equations and the Compute Engine to perform symbolic computing and numeric evaluations in JavaScript environments.

These tools are available under an Open Source license.

I can’t wait to see what you do with them. Enjoy!
